📚 RAG Chatbot using LangGraph, ChromaDB & Groq

📌 Project Overview

This project implements a Retrieval-Augmented Generation (RAG) Chatbot that answers questions from PDF documents using Large Language Models (LLMs). The chatbot retrieves relevant document chunks from a vector database and generates accurate, context-aware responses.

---

🛠️ Tech Stack

- Python
- LangGraph
- ChromaDB
- Hugging Face Embeddings
- Groq LLM
- Google Gemini API
- Streamlit
- Google Colab
- GitHub
