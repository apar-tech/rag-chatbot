## Project Overview
This project implements a Retrieval-Augmented Generation (RAG) chatbot using:
- PDF documents
- Hugging Face Embeddings
- ChromaDB Vector Database
- Groq LLM

## Workflow

PDF
↓
Chunking
↓
Embeddings
↓
Vector Database
↓
Retrieval
↓
Groq LLM
↓
Answer

## Technologies Used
- Python
- LangChain
- ChromaDB
- Groq
- Streamlit
